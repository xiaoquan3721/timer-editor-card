# 定时编辑器卡片 (Timer Editor Card)

一个为 Home Assistant 定制的定时自动化编辑器卡片，采用内嵌折叠面板式 UI，支持定时和倒计时两种模式。

## 效果预览

- 白色圆角卡片，内嵌配置表单
- 橙色渐变强调色
- 折叠面板：副实体 / 图标 / 功能 / 自动化定时
- 2 列网格表单布局
- 底部固定操作栏（代码编辑器 / 取消 / 保存）
- 任务列表带 Toggle 开关

## 安装步骤

### 第一步：上传文件

将 `timer-editor-card.js` 上传到你的 HA 配置目录：

```
config/
└── www/
    └── timer-editor-card/
        └── timer-editor-card.js
```

### 第二步：注册资源

打开 Dashboard → 右上角菜单 → **管理资源** → **添加资源**

```
URL: /local/timer-editor-card/timer-editor-card.js
资源类型: JavaScript Module
```

或者手动在 `configuration.yaml` 中添加（**注意缩进使用空格**）：

```yaml
lovelace:
  resources:
    - url: /local/timer-editor-card/timer-editor-card.js
      type: module
```

> **重要**：YAML 对缩进敏感，必须用空格而非 Tab，否则会导致恢复模式！

### 第三步：添加卡片

编辑 Dashboard，添加卡片：

```yaml
type: custom:timer-editor-card
title: 定时编辑
icon: mdi:clock-outline
```

### 第四步：清除缓存（关键！）

**如果更新后界面没有变化，请执行以下操作：**

1. **浏览器强制刷新**：按 `Ctrl+F5`（Windows）或 `Cmd+Shift+R`（Mac）
2. **手机端**：进入 HA App → 设置 → 伴侣应用 → 调试 → 清除前端缓存
3. **或使用无痕模式**打开 HA 验证
4. **终极方案**：在浏览器地址栏访问 `YOUR_HA_URL/local/timer-editor-card/timer-editor-card.js?v=2.0.0` 确认是最新代码

## 使用说明

### 创建定时任务

1. 在卡片中展开"自动化定时"折叠面板
2. 打开橙色 Toggle 开关启用定时功能
3. 选择场景（可选）
4. 输入主实体名称
5. 选择执行动作（关闭/开启/切换）
6. 选择重复周期（单次/每天/工作日/周末）
7. 设置小时和分钟
8. 点击"保存"

### 倒计时模式

1. 在"自动化定时"面板中，打开"倒计时模式"开关
2. 设置倒计时时长（时/分/秒）
3. 选择执行动作
4. 点击"保存"后立即启动倒计时

### 编辑/删除任务

- 点击列表中的任务可编辑
- 编辑界面有红色删除按钮

### 高级功能

- **执行条件**：可选择"无条件 / 仅夜间 / 仅白天"
- **单次任务**：勾选"执行后自动停用"
- **代码编辑器**：点击底部"<> 代码编辑器"查看/编辑 YAML

## 故障排查

### 更新后界面没变化

这是最常见的问题，原因是浏览器缓存了旧版本 JS。

**解决方案：**
1. 在 `configuration.yaml` 中修改资源 URL 添加版本号：
   ```yaml
   lovelace:
     resources:
       - url: /local/timer-editor-card/timer-editor-card.js?v=2.0.0
         type: module
   ```
2. 然后 **Configuration → Server Controls → Reload UI**
3. 浏览器按 `Ctrl+F5` 强制刷新

### 卡片不显示

1. 确认 JS 文件路径正确
2. 确认资源已注册
3. 检查浏览器控制台是否有红色报错

### 保存失败

1. 确认选择了设备/填写了实体名称
2. 检查时间设置是否正确
3. 确认用户有创建自动化的权限
